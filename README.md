## Setup Guide

1. Install the Dependencies
    ```bash
    bun install 
    ```

2. Rename .example.env to .env & replace YOUR_TMDB_API_KEY to your actual TMDB api key

3. Run the Application
    ```bash
    bun run dev
    ```
